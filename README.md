# CubicPostcode

The open, precise, 3D universal addressing system.

More professional, free, and collaborative than what3words.

## Features
- 1m³ resolution
- Cross-platform apps
- Fully open source

Live site: https://cubicpostcode.github.io/cubicpostcode/
